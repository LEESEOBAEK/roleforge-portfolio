# Demo Assets

This folder is reserved for future animated README and GitHub Pages demos.

Recommended files:

- `roleforge-demo.gif` - 10 to 20 second README hero demo.
- `roleforge-demo.mp4` - higher quality GitHub Pages demo.
- `roleforge-demo-poster.png` - static poster image for video fallback.

Recommended flow:

1. Open the workspace overview.
2. Switch to Agent Registry.
3. Open Document Intake.
4. Show Release Safety.
5. End on the public-safe portfolio boundary.

Capture guidance:

- Browser width: 1440 px or wider.
- Browser height: 1000 px or taller.
- Browser zoom: 80% or 90%.
- Hide the chat widget before recording.
- Keep each interaction slow enough to read.
